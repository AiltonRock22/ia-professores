## 📖 Sobre o Projeto

Este é um portal interativo no formato **Single Page Application (SPA)** construído especialmente para guiar professores do Ensino Fundamental II, Ensino Médio Integral e EJA no uso ético, produtivo e crítico das principais ferramentas de Inteligência Artificial disponíveis no mercado.

O projeto foi desenvolvido pelo **Prof. Ailton** (Filosofia & Informática) como parte de um mini-curso prático para capacitação docente.

## ✨ Funcionalidades

| Funcionalidade | Descrição |
|----------------|-------------|
| 🎯 **Painel Personalizado** | Apresentação do mini-curso com informações do professor e da escola |
| 🤖 **Guias por IA** | Páginas dedicadas para ChatGPT, Gemini, Copilot, NotebookLM, DeepSeek e Manus IA |
| 📋 **Copiador de Prompts** | Botão com um clique para copiar prompts educacionais otimizados (com feedback visual) |
| ⚖️ **Módulo de Ética** | Orientações sobre LGPD, privacidade e novas abordagens avaliativas |
| 🤝 **Espaço Colaborativo** | Formulário para professores compartilharem suas próprias práticas com IA (armazenamento local) |
| 📱 **Design Responsivo** | Adaptado para celulares, tablets e desktops |

## 🛠️ Tecnologias Utilizadas

- **HTML5** - Estrutura semântica
- **Tailwind CSS** - Estilização moderna e responsiva
- **Lucide Icons** - Ícones vetoriais
- **Vanilla JavaScript** - Lógica SPA, navegação entre abas, sistema de cópia e armazenamento local
